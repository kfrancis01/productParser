Control popup information

When this is switched on the popup box will have a paw print icon to indicate that the product is cruelty-free. Otherwise the popup box will not have the paw print icon.

When this is switched on the popup box will have a vegan icon if the product is vegan; otherwise the icon will not show up.

The popup border will turn red if the product is found to not be eco-friendly.

A smiley face will be added to the popup if this product will not break you out.
